Login:
    login&&username&&password
will return:
    loginAcc||KeyStr

game session request:
    gameReq&&key&&targetUsername
if the target user is not in connection, server will send:
    reqDec||User not online

Accept a request:
    accReq&&key&&senderUsername
Server will send to the two parties in the request:
    reqAc||opponentUsername

When receiving a request, server will send:
    recReq||senderUsername||senderDeckCount

Server warning:
    err||message
